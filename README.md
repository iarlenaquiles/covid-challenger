# :computer: API em NodeJS que usa a API dos dados da COVID19 e mostra o top 10 das cidades de um estado passado por parâmetro e um período

### Endpoint
 - `covid?state=CE&dateStart=2020-05-10&dateEnd=2020-05-20`
## Tecnologias

 - NodeJS
 - Express
 - Nodemon
 - date-fns

 
 
## Run
 - git clone
 - yarn
 - yarn dev
